# ИИ-ассистент по Excel

## Описание
Streamlit-приложение для анализа Excel-файлов (листы: Заявки, Заказы, Предприятия) с помощью ИИ на основе OpenAI GPT-5.2.

## Архитектура

- **app.py** — главный Streamlit-интерфейс (3 вкладки: Чат, Данные, Примеры вопросов)
- **data_processor.py** — загрузка, парсинг и схемирование Excel (openpyxl + pandas)
- **ai_assistant.py** — генерация Python-кода через OpenAI API, безопасное выполнение, возврат text/df/fig
- **attached_assets/Сентябрь_2020_1773221143241.xlsx** — тестовый файл с 3 листами

## Ключевые функции
- Автоматическая загрузка тестового Excel при старте
- Загрузка пользовательских .xlsx/.xls файлов
- Многоходовый диалог (контекст последних 8 вопросов)
- Генерация текстового ответа + таблицы + matplotlib-графика
- Экспорт результатов: CSV, XLSX, PNG
- Просмотр сгенерированного кода (опционально)
- 10 готовых примеров вопросов

## Зависимости
- streamlit, pandas, openpyxl, matplotlib, plotly, kaleido, fpdf2, xlsxwriter
- OpenAI через Replit AI Integrations (без собственного API-ключа)

## Запуск
```bash
streamlit run app.py --server.port 5000
```

## Контекст данных (тестовый файл — сентябрь 2020)
- **Заявки**: 2440 строк, 12 колонок. Ключевые: Дата заявки, Номер, Клиент, Сумма, Менеджер ОЗ, Заказано
- **Заказы**: 503 строки, 26 колонок. Ключевые: Дата, Клиент, Сумма, Менеджер ОП, Подразделение
- **Предприятия**: 10 строк (справочник), Код, Наименование, Регион

## Кодирование подразделений (по префиксу номера заявки)
- SPB → Санкт-Петербург (центральный офис)
- MSK → Москва
- EKB → Екатеринбург
- SMR → Самара
- KRD → Краснодар
- KZN → Казань
