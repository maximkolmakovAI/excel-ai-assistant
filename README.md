# Excel AI Assistant

Streamlit-приложение для анализа Excel-данных с помощью ИИ.

## Secrets

Добавьте в Settings → Variables and secrets:
- `PROXY_API_KEY` - ваш API ключ
- `PROXY_BASE_URL` - URL прокси
